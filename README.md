#### Getting Started:

Install Dependencies

```bash
$ npm i
```

Build Production

```bash
$ npm run build
```

Whip up dev server

```bash
$ npm run dev
```

#### E2E Testing (Cypress):

Open Cypress

```bash
$ npx cypress open
```

-   launch e2e suite in browser of your choice
